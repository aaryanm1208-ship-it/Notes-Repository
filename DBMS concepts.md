
# DBMS & SQL Quick Cheatsheet

## DBMS Concepts
- Primary Key: unique identifier
- Foreign Key: reference to another table
- Normalization: reduce redundancy
- Indexing: faster search

## SQL Commands
SELECT * FROM table;
INSERT INTO table VALUES();
UPDATE table SET col=value;
DELETE FROM table;

## Constraints
- NOT NULL
- UNIQUE
- PRIMARY KEY
- FOREIGN KEY

## Joins
- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- FULL JOIN

## Aggregation
- COUNT()
- SUM()
- AVG()
- MAX()
- MIN()

## GROUP BY & HAVING
GROUP BY column
HAVING condition

## ACID Properties
- Atomicity
- Consistency
- Isolation
- Durability
