
# Java Quick Cheatsheet

## Basics
- JVM → JRE → JDK
- main(): public static void main(String[] args)

## OOP Concepts
- Encapsulation: private fields + getters/setters
- Inheritance: extends
- Polymorphism: method overriding
- Abstraction: abstract class / interface

## Keywords
- final: constant / cannot override / cannot inherit
- static: class-level
- this: current object
- super: parent reference

## Collections
- ArrayList: dynamic array
- LinkedList: fast insert/delete
- HashMap: key-value (no order)
- HashSet: unique values

## Exception Handling
try { }
catch(Exception e) { }
finally { }

## File Handling
- File
- FileReader / FileWriter
- BufferedReader

## Important Tips
- Prefer interfaces
- Use equals() not == for objects
